========================================
HOW TO USE THIS FILE
========================================

1. Fill in your API keys above
2. Save this file securely (Google Drive, 1Password, etc.)
3. On Raspberry Pi, copy values to .env file:
   
   cd ~/trading/options-ai-bot
   nano .env
   
4. Paste the values (only the lines with = signs)
5. Save: Ctrl+X, Y, Enter

========================================
GETTING API KEYS
========================================

ALPACA (Trading):
1. Go to https://alpaca.markets/
2. Sign up for free account
3. Go to "Paper Trading" section
4. Copy API Key and Secret Key

DISCORD (Bot):
1. Go to https://discord.com/developers/applications
2. Create "New Application"
3. Go to "Bot" section
4. Click "Add Bot"
5. Copy Bot Token
6. Enable "Message Content Intent"
7. Invite bot to your server
8. Copy Channel ID (right-click channel → Copy ID)

OPENAI (AI):
1. Go to https://platform.openai.com/
2. Sign up / Login
3. Go to API Keys section
4. Create new secret key
5. Copy the key (save it - can't see again!)

ANTHROPIC (Optional):
1. Go to https://console.anthropic.com/
2. Sign up / Login
3. Go to API Keys
4. Create new key

NEWS API (Optional):
1. Go to https://newsapi.org/
2. Sign up for free account
3. Copy API key

========================================
SECURITY TIPS
========================================

✅ DO:
- Save this file in encrypted storage (Google Drive, 1Password)
- Use different API keys for paper vs live trading
- Rotate keys periodically
- Keep .env file in .gitignore (already done)

❌ DON'T:
- Share API keys with anyone
- Commit .env to git
- Post keys in Discord/forums
- Use live trading keys for testing
